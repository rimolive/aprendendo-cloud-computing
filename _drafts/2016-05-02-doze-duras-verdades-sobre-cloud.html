Desempenho não uniforme em Máquinas Virtuais

O primeiro item que o autor relata é a não uniformidade de desempenho, onde ele relata que de acordo com seus benchmarks máquinas diferentes com mesma quantidade de RAM e sistema Operacional se comportam de forma diferente. Apesar do autor não revelar em detalhes seus testes de benchmark, vamos primeiro discutir a respeito do item Desempenho em Cloud Computing.
Para quem usa AWS, Goocle Cloud Platform ou Kubernetes já deve ter ouvido falar do ECU (EC2 Compute Unit), GCEU (Google Compute Engine Unit) ou até mesmo no KCU (Kubernetes Compute Unit), que são algoritmos de normalização de CPU que permitem chegar a uma unidade de medida baseada em todos os recursos computacionais que o provedor oferece. Assim, é possível ter uma forma consistente de quanto de CPU as aplicações utilizam.
