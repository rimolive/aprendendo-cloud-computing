O Node.js

O Node.js é uma plataforma Javascript construída em cima da implementação do Chrome de forma a construir aplicações usando diversas funcionalidades tais como eventos, um modelo de I/O diferente que o torna mais eficiente em aplicações em tempo real e com uma grande quantidade de dados.
Uma das grandes vantagens de se usa o Node.js na minha opinião é simplesmente porque Javascript já é muito conhecida por ser um dos pilares da World Wide Web. Sempre que pensamos em www, já passa pelas nossas cabeças a tríade HTML + CSS + JS e por isso torna-se melhor desenvolver aplicações dessa forma.
